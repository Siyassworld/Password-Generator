Project Overview

This Java project creates a password generator that generates strong , unique password based on user-selected character lengths(6,10 or 12 digits). If no option is selected, it defaults to an 8-character password.

Features

	• Generates password with a mix of uppercase and lowercase letters, numbers, and special characters.
	• Allows users to select password length(6,10 or 12 characters).
	• Defaults to 8-character password if no option is selected.
	• Ensures password complexity and randomness.

Usage

	1. Run the PasswordGenerator class.
	2. Select the desired password length (6,10 or 12 characters) or press enter for default.
	3. The generated password will be displayed.

Code Structure

	• PasswordGenerator.java : Main class containing password generation logic.

Requirements

	• Java 8 or higher.
	• No external dependencies required.
